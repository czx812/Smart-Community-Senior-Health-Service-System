# 智慧社区老人健康服务平台
本项目为社区老人健康管理系统，面向社区居家养老场景，实现老人健康数据监测、异常预警、健康档案管理与服务对接。系统帮助社区工作人员高效管理老人健康状况，提升养老服务效率与安全性。
1. 基础管理类

用户管理模块：管理系统内所有用户（老人、家属、管理员、医护人员等）的账号、权限和信息。
老人档案管理模块：建立和维护老人的完整档案，包括基本信息、健康史、家属联系方式等，方便统一管理。

2. 健康监测与慢病管理类

健康数据采集与监测模块：实时或定期采集老人的健康数据（如血压、血糖、心率等），并进行持续监测。
慢病管理与随访模块：针对高血压、糖尿病等慢性病，提供病程跟踪、用药指导和定期随访服务。

3. 提醒与服务类

用药 / 体检 / 服务提醒模块：通过打卡吃药、帖子发表、问题询问等方式，提醒老人按时用药、体检或接受其他服务。
预约与上门服务模块：支持老人或家属在线预约体检、护理、家政等服务，并可申请上门服务。

4. 安全与宣教类

紧急求助与预警模块：老人可一键求助，系统同时标记高风险老人信息，及时向家属和管理员推送预警。
健康知识与宣教模块：通过组织老人集体活动、推送健康知识（如饮食建议）等方式，进行健康宣教。

5. 数据分析类

统计分析与报表模块：对健康数据、服务记录等进行统计分析，生成各类报表，辅助决策和管理。
